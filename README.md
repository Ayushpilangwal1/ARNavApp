# ARNavApp

An Augmented Reality Navigation App using ARCore.

## 🚀 Features
- Real-time AR navigation
- OpenGL rendering
- Directional guidance
- Location-based tracking

## 🛠 Tech Stack
- Java / Kotlin
- ARCore
- OpenGL
- Android SDK

## 📌 Description
ARNavApp helps users navigate using augmented reality by overlaying directions on the real world.
